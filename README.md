# previsao_tempo

App simples de previsão do tempo feito em Flutter.

## Como rodar

1. Instale as dependências:

   ```powershell
   flutter pub get
   ```
2. Rode o app no navegador:

   ```powershell
   flutter run -d chrome
   ```

## O que o app faz

- Salva a última localização buscada e recarrega ao abrir.
- Busca o clima em tempo real pela API do OpenWeatherMap.
- Mostra temperatura atual, mínima, máxima, humidade e velocidade do vento.
- Permite consultar o clima filtrando por estado e cidade.
